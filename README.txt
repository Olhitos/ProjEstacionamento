APP SISTEMA DE ESTACIONAMENTO:
proposta: o aplicativo tem por objetivo implementar um sistema de gerenciamento para um estacionamento, usando a linguagem java ,
bem como os conceitos apresentados durante a disciplina de lp2

funcionamento:
- o sistema de interface gráfica é implementado na main, conectando as funções das diversas classes contidas nesse projeto
- cada tipo de janela na interface possui uma classe correspondente, e os objetos descritos no documento enviado pelo professor também 
  foram implementados conforme recomendado
- o sistema permite: 
  - cadastro de gerente
  - cadastro de cliente (veiculo)
  - reserva de vagas - carro
  - reserva de vagas - moto
  - reserva de vagas - caminhão
  - pagamentos utilizando os meios de pagamento sugeridos (dinheiro,pix, cartão)
  - desocupação da vaga e consequente reocupação por outros veiculos

execução:
- o sistema é executado sobre ambientes que suportem versões recentes do JDK (java development kit), seja uma IDE própria (vscode,intelij,eclipse,etc..) ou 
  pelo terminal do sistema
- caso a janela do menu principal seja fechada, a execução do programa é encerrada imediatamente

considerações finais:
- o trabalho ajuda a compreender melhor como as ferramentas da linguagem java, e da orientação a objetos, se comunicam entre si em um projeto de escopo maior,
  e ressaltam a ideia de trabalhar em equipe no projeto, dado a independencia das partes do código, e facilidade de expansão dos mesmos
